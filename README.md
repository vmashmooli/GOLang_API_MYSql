# GOLang_API_MYSql

Simple MySql management with API project in GOLang

## Description

Create Simple MySql management like add records, remove, edit and serach with ECHO API and GORM in GOLang

## Getting Started

### Dependencies

* ECHO
* GORM
* MySql
* ubuntu 22.4


### Executing program

* go run server.go
